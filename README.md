# OOP POS Management System

## Description

This project was developed as part of the Object-Oriented Programming course using C# and .NET 6. The application represents a Point of Sale (POS) management system designed to manage products, services and commercial packages while demonstrating the practical application of Object-Oriented Programming principles.

The system allows users to create, manage and organize products and services, combine them into commercial packages, perform filtering and sorting operations, execute LINQ queries and persist data using XML serialization and deserialization mechanisms.

The main objective of the project was to implement and integrate advanced OOP concepts in a realistic business scenario. The application architecture is based on abstraction, inheritance, polymorphism and interfaces, ensuring modularity, extensibility and code reusability.

## Features

* Product management
* Service management
* Commercial package creation
* Sorting packages by price
* Filtering by category
* Filtering by maximum price
* LINQ-based queries
* XML serialization and deserialization
* Generic collections implementation
* Modular object-oriented architecture

## Technologies Used

* C#
* .NET 6
* LINQ
* XML Serialization
* Visual Studio

## OOP Concepts Implemented

* Classes and Objects
* Abstract Classes
* Inheritance
* Interfaces
* Polymorphism
* Generics
* Encapsulation
* Separation of Concerns

## Project Structure

The solution is divided into multiple components responsible for entity definitions, business logic, filtering mechanisms and application management. Products, services and packages are represented through dedicated classes, while filtering and querying functionalities are implemented through interfaces and reusable components.

## Learning Outcomes

Through this project, I gained practical experience in designing object-oriented applications, applying software design principles, implementing reusable components and managing data persistence using XML technologies.

# Sistem POS pentru Gestionarea Produselor și Serviciilor (RO)

## Descriere

Acest proiect a fost realizat în cadrul disciplinei Programare Orientată pe Obiecte utilizând C# și .NET 6. Aplicația reprezintă un sistem de gestionare de tip Point of Sale (POS), conceput pentru administrarea produselor, serviciilor și pachetelor comerciale, având ca scop demonstrarea practică a conceptelor fundamentale de Programare Orientată pe Obiecte.

Sistemul permite crearea, gestionarea și organizarea produselor și serviciilor, gruparea acestora în pachete comerciale, efectuarea operațiilor de filtrare și sortare, realizarea interogărilor LINQ și persistarea datelor prin mecanisme de serializare și deserializare XML.

Obiectivul principal al proiectului a fost integrarea conceptelor avansate de OOP într-un scenariu apropiat de o aplicație reală de business. Arhitectura aplicației se bazează pe abstractizare, moștenire, polimorfism și interfețe, asigurând modularitate, extensibilitate și reutilizarea codului.

## Funcționalități

* Gestionarea produselor
* Gestionarea serviciilor
* Crearea pachetelor comerciale
* Sortarea pachetelor după preț
* Filtrarea după categorie
* Filtrarea după preț maxim
* Interogări bazate pe LINQ
* Serializare și deserializare XML
* Implementarea colecțiilor generice
* Arhitectură modulară orientată pe obiecte

## Tehnologii utilizate

## Concepte OOP implementate
